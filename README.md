# Estoque Roupas App

Aplicativo mobile desenvolvido para controle de estoque de loja de roupas.

## Tecnologias

- React Native
- Expo
- TypeScript
- SQLite
- Expo Router

## Funcionalidades

- Cadastro de produtos
- Controle de estoque
- Entrada e saída de produtos
- Histórico de movimentações
- Dashboard gerencial
- Persistência local de dados
